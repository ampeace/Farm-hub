# Farm-Hub

This is the Expo version of the application, which ensures all facilities that will help farmers grow in the market.

## How to Run the Project Locally

To get started with the project on your local machine, follow these steps:

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine
- [Expo CLI](https://docs.expo.dev/get-started/installation/) installed globally (`npm install -g expo-cli`)
- For Android: [Android Studio](https://developer.android.com/studio) or an Android device with the Expo Go app.
- For iOS: [Xcode](https://developer.apple.com/xcode/) (macOS only) or an iOS device with the Expo Go app.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd Farm-hub
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Run on specific platforms:**
   - Press `a` in the terminal to open in Android emulator.
   - Press `i` in the terminal to open in iOS simulator.
   - Press `w` in the terminal to open in a web browser.
   - Alternatively, scan the QR code generated in the terminal using the Expo Go app on your physical device.
